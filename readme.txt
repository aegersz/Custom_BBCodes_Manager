[color=blue][b][size=12pt][u]Introduction[/u][/size][/b][/color]
This MOD adds the ability to create custom BBCode for your forum.  

[color=blue][b][size=12pt][u]Admin Settings[/u][/size][/b][/color]
Settings are available on the [b][i]Admin[/i] -> [i]Forum[/i] -> [i]Posts and Topics[/i] -> [i]Custom BBcode[/i][/b] page.

To add a custom tag, click the "Create New Tag".  Once you submit the tag, you can go back and upload a button for it.  The image should be 23 x 22 GIF with transparent background and no larger than 10kb.

[color=blue][b][size=12pt][u]Compatibility Notes[/u][/size][/b][/color]
This mod was tested on SMF 2.0.4 and 2.0.5.  SMF 1.x is not and will not be support by this mod.

This mod makes only two edits to the sources, one to the [i]Admin.english.php[/i] and one to [i]ManagePosts.php[/i].  This mod makes use of SMF 2.0 hooks as much as possible.  It should work on all versions of SMF 2.0.
